A sweet-toothed CSS theme with retro summer vibes.

What's this then?

Sundae Times used to be a Tumblr theme, back when I made that sort of thing. Now it's a stylesheet. It's not a design system in the truest sense; it's just a set of styles for pure HTML elements, with a few utility classes and variants for good measure.

Come to think of it, it's probably still useful as a Tumblr theme.

Can I use it?

Sure, if it's for personal use only. I'd appreciate a credit if you do. However, this theme does not pass accessibility colour contrast tests, so it is unsuitable for professional use.
