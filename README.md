# Sundae Times

Sundae Times is a visual theme stylesheet, based on my previous work in Tumblr themes.

## Usage

Add the stylesheet in the `<head>` section of your HTML:

```html
<link rel="stylesheet" href="https://dentednerd.github.io/sundae-times/index.css" />
```

## Features

- styles for many (but not all) HTML elements, including `<article>`
- a set of CSS classes for quick visual theming
- CSS custom properties, based on Sass variables

## Development

`npm run dev` runs Sass in `--watch` mode.
